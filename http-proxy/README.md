# HTTP Proxy

## Instructions

Make a program capable of proxying http requests made to it.

## Key Features

- configurable to act as a real proxy or to expect target `url` as a query param

## Usage

```shell
go run http-proxy/main.go [-p=8003] [-url=true]
```
